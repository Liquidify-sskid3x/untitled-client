Untitled Client

Utility client for Minecraft (Java Edition) 1.21.11 using Fabric. Inspired by Meteor Client.

Combat:

KillAura: Attacks nearby entities
TriggerBot: Attacks on crosshair contact
AimAssist: Adjusts aim toward targets
Criticals: Forces critical hits
Reach: Extends hit range

Movement:

Fly: Allows flight
Jesus: Walk on water
Speed: Increases movement speed

Player

AirPlace: Places blocks in air
AutoArmor: Equips best armor
AutoTotem: Holds totem automatically
NoFall: Prevents fall damage
Scaffold: Places blocks while moving
SafeWalk: Prevents edge falls
Velocity: Modifies knockback

Render

ESP: Highlights entities
